SDK Extensions new

session-lifecycle

security hooks
🛡

Lifecycle management extension with resource monitoring, security guards, and automatic session logging to Obsidian.

onSessionStart
Runs resource check (CPU, hung processes, stale Docker containers, orphaned Python). Alerts if issues found.
onPreToolUse
Security guard — blocks 7 dangerous command patterns: rm -rf /, sudo rm, chmod -R 777, git push --force (allows --force-with-lease), curl|bash, wget|bash.
onPostToolUse
Reminds about dual ruff check when Python files are modified in vuln-agg repos.
onSessionEnd
Logs session summary to Obsidian vault via log-session.sh, triggers QMD reindex. Skips trivial sessions (<3 turns, <120s).
onErrorOccurred
Auto-retries recoverable model call failures (up to 2 retries).
